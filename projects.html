Ask485

Ask485 is project my team and I worked on for my web development class. This project required us to build a search engine for a collection of Wikipedia articles using an inverted index and PageRank.

The project consisted the three parts. In the first part, we used a MapReduce framework to generate an inverted index for the collection of documents. We then created a Rest API to request and combine data from three inverted index servers storing the inverted index we computed in part one. Lastly, we made a Flask server which dynamically rendered pages to the user based on their query.

Insta485

Insta485 is a recreation of Instagram I made for my web development class with my project team. Using React, we created a client-side dynamic page which rendered a unique page based on who the user follows. We also implemented several interactive features such as like and comment buttons which worked without refreshing the page.

For the back end, we designed a Rest API we made asynchronous requests to, allowing our frontend to access data from our SQL database. Lastly, we used server-side dynamic pages rendered with Flask to create pages for each user.
